Quick IP Lookups with Browser Extensions

Browser Extension August 2025

In cybersecurity and network administration, speed matters. When you encounter an unfamiliar IP address in logs, emails, or web content, every second spent switching between tabs and copying addresses is time that could be spent analyzing threats. The Who Dis? IP Lookup Extension eliminates this friction entirely.

The Workflow Problem

The traditional IP lookup process is surprisingly tedious:

  1. Spot an IP address on a webpage
  2. Select and copy the IP
  3. Open a new tab
  4. Navigate to an IP lookup service
  5. Paste the IP and search
  6. Wait for results

For security professionals who might investigate dozens of IPs daily, this process becomes a significant productivity drain.

The Right-Click Solution

Who Dis? simplifies this entire workflow to a single right-click. When you encounter an IP address on any webpage, simply right-click on it and select "Look up IP address" from the context menu. The extension automatically detects both IPv4 and IPv6 addresses and opens the lookup results in a new tab.

"The best security tools are the ones that get out of your way and let you focus on the analysis, not the process."

Security-First Design

✅ What Works Well

  • Look up IPs easier - Streamlined workflow saves time and reduces context switching
  • Preventing complacency - Quick lookups encourage regular IP verification instead of ignoring unknown addresses
  • IPv4 & IPv6 support - Handles both address formats seamlessly
  • Zero data collection - Extension processes everything locally
  • Security focused - Minimal permissions and transparent operation

⚠️ Limitations

  • Relies on third party website - Dependent on external IP lookup service availability
  • Specific URL format dependency - Changes to the lookup service could break functionality

Technical Implementation

Built using Chrome's Manifest V3 specification, the extension uses the contextMenus API to add the lookup option when right-clicking on selected text. The extension validates the selected text against IP address patterns before showing the context menu option, ensuring it only appears when relevant.

The privacy-conscious design means no IP addresses are logged or transmitted through the extension itself—all lookups go directly to the third-party service.

Beyond Basic Lookups

While the current version focuses on simplicity, future enhancements could include:

  • Multiple lookup service options
  • Cached results for recently queried IPs
  • Integration with threat intelligence feeds
  • Bulk IP analysis capabilities

The Bigger Picture

Who Dis? represents a philosophy of security tooling: reduce friction to increase security practice adoption. When security tasks are easy and fast, people are more likely to perform them regularly. By eliminating the small barriers to IP investigation, we encourage a more security-conscious browsing and analysis workflow.

The extension works perfectly for network administrators, security professionals, and anyone who regularly encounters IP addresses in their digital work and wants to maintain awareness of their network environment.
